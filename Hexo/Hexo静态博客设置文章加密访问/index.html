Hexo静态博客设置文章加密访问

引言

总有一些想法，自己想说出来，但又找不到人倾诉。别人不一定能理解你，你也害怕受到别人异样的目光。

于是你发布在博客上面，但是自己有时候请同学室友来看自己的博客，这样又会造成尴尬。你只想把这些话，说给你觉得值得的人听，所以接下来这个插件，可以轻松实现文章的加密功能，让自己的想法只展示给懂你的人看。👀

安装hexo-blog-encrypt插件

在hexo目录下

npm install hexo-blog-encrypt

使用插件

在想要使用加密功能的Blog头部加上对应文字：

  • abstract：用于设置加密文章显示的摘要。
  • message：输入密码提示。
  • wrong_pass_message: 输入错误提示。
