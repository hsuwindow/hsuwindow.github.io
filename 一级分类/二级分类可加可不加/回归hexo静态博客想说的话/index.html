自己的权衡和考量

试过hexo + github + typora + picgo

试过wordpress搭建

试过notion自带的博客生成

感觉还是不够好,wordpress自己买服务器买域名很贵,,notion自带的博客生成很卡

试过hexo + github + typora + picgo本地也有,但是之前的那个站点被我搞的样子太丑,,改回来太麻烦,,重新弄一个博客好像也没必要.

要我说还得是wordpress自己买服务器买域名,咋毕竟是程序员,不折腾点真正的网络怎么行?

但是之前都是在ubuntu下进行的

而且费用问题还是要精打细算

  1. 之前的服务器和域名能不能找回来
  2. 找不回来的话重新搭建一个费用能不能接受
  3. 重新搭建是不是ubuntu和windows都能是上传和修改文件

突然想起来,我是不是重装过系统,那么我的ubuntu还有那些配套的软件环境其实是没有的

那我还是回归

hexo + github + typora + picgo本地有一份,github上也有一份,相当于github做了一次服务器.

比notion好,比worpress好,

看看能不能双端通用

和我想的一样,直接用pull指令就可以做到同步,没想到第一次体验到的github上面的合作的行为竟然是自己的双系统,,这样也好,,有这样的一片博客支撑,那我就先在windows上面安装吧

https://cloud.tencent.com/developer/article/1833963

https://cloud.tencent.com/developer/article/1833961

https://cloud.tencent.com/developer/column/92114

而且还能windows和ubuntu都可以用

对于我说的typora绑定上传图片的问题,

我可以暂时用别的例如marktext和word来记录呀

反正你就把typora和写博客绑定起来就好了

博客上还是写学习技术相关的 ,别再把自己的一些负面情绪搞在上面,那样博客很乱,,这些贴近生活的你就放在notion上面

win10部署hexo+githubpages

红温了,感觉我的github也没啥好东西,,看了一下,确实都是垃圾

直接再创建一个账号得了

伟大的博客,解决了我的所有问题.直接照着他来

hexo+github+腾讯云+typora零费用搭建个人博客

我下载的picgo版本是2.3.1的稳定版

安装nodejs

https://nodejs.org/dist/latest/

我下载的版本是node-v23.0.0-x64.msi

安装git

切记要用管理员方法打开cmd

运气真好,成功了

用之前已经实名过的阿里云账号买域名得了

以后都用管理员模式进入win的命令行

他那个bat我懒得动,我担心ubuntu出问题

YES!!!!!
